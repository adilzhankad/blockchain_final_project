
## Project Structure

```

contracts/
RewardToken.sol              # ERC-20 reward token (minted for contributions)
Crowdfunding.sol             # Crowdfunding logic: campaigns, contributions, mint tokens
scripts/
deploy.js                    # Deploy token + crowdfunding and set token minter
hardhat.config.js              # Hardhat config + localhost network

````

---

## Requirements
- Node.js (recommended: 18+)
- npm
- Git

---

## Install

```bash
npm install
````

---

## Compile

```bash
npx hardhat compile
```

---

## Run local blockchain

Open a separate terminal:

```bash
npx hardhat node
```

This runs a local Ethereum network at `http://127.0.0.1:8545` (chainId 31337).

---

## Deploy contracts (localhost)

In the first terminal:

```bash
npx hardhat run scripts/deploy.js --network localhost
```

After deploy, you will see:

* Token deployed to: 0x...
* Crowdfunding deployed to: 0x...
* Minter set 

Save these addresses for testing.

---

## Test in Hardhat console

```bash
npx hardhat console --network localhost
```

Paste commands (replace addresses with your deployed ones):

```js
const [owner, alice] = await ethers.getSigners();

const token = await ethers.getContractAt("RewardToken", "TOKEN_ADDRESS");
const crowd = await ethers.getContractAt("Crowdfunding", "CROWD_ADDRESS");

// create campaign (goal 1 ETH, duration 1 hour)
await crowd.createCampaign(
  "Aid shipment #1",
  "Humanitarian delivery: medicines",
  ethers.parseEther("1"),
  3600
);

// contribute 0.1 ETH from alice
await crowd.connect(alice).contribute(0, { value: ethers.parseEther("0.1") });

// check token balance (should be > 0)
await token.balanceOf(alice.address)

// pretty format
ethers.formatEther(await token.balanceOf(alice.address)) // expected: 10.0 if 1 ETH = 100 tokens
```
